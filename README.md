# 开发初衷
* 博足ERP立志为中小企业提供开源好用的ERP软件，降低企业的信息化成本
* 个人开发者也可以使用博足ERP进行二次开发，加快完成开发任务
* 初学JAVA的小伙伴可以下载源代码来进行学习交流

# 技术框架
* 核心框架：SpringBoot 2.0.0
* 持久层框架：Mybatis 1.3.2
* 日志管理：SLF4J 1.7
* 前端框架：Vue 2.7.16
* UI框架: Ant-Design-Vue 1.5.2
* 模板框架: Jeecg-Boot 2.2.0
* 项目管理框架: Maven 3.3.9

# 开发环境
建议开发者使用以下环境，可以避免版本带来的问题
* IDE: IntelliJ IDEA 2025.1.4.1
* DB: Mysql 8.0.24
* JDK: JDK 1.8
* Node: Node 20.17.0
* Maven: Maven 3.3.9
* Redis: 6.2.1
* Nginx: 1.12.2 

# 服务器环境
* 数据库：Mysql8.0.24
* JAVA平台：JRE1.8
* Redis库：redis6.2.1
* Nginx代理：nginx1.12.2
* 操作系统：Windows、Linux等
